# 🍄 Mushroom Classification (End-to-End ML Project)

This project focuses on classifying mushrooms as **edible or poisonous** using machine learning techniques.

## 🚀 Project Overview
- Built an end-to-end ML pipeline
- Data preprocessing and feature engineering
- Model training and evaluation
- Achieved **99.28% accuracy**

## 📊 Dataset
- Mushroom dataset with categorical features
- Cleaned and encoded for model training

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Jupyter Notebook

## 📈 Results
- Accuracy: **0.99288**
- High precision and recall

## 🧠 Key Learnings
- Handling categorical data
- Feature encoding techniques
- Model evaluation and tuning
